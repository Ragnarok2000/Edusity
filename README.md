# Edusity

Project using **React JS + CSS** (Vite)  
This is a pure front-end project—fully responsive for all devices.

---

## About

Edusity is a modern educational website template built with React and styled with CSS, bootstrapped via Vite for fast performance.

---

## Features

- Built with React + Vite for blazing-fast development and builds
- Modern, clean responsive design
- Ready to deploy (Vercel/Netlify/Static host)
- Modular and reusable React components

---

## Getting Started

Clone this repository and install dependencies:
git clone https://github.com/Ragnarok2000/Edusity.git
cd Edusity
npm install
npm run dev


- `npm run dev` — Starts the local dev server
- `npm run build` — Builds for production (output in `dist`)
- `npm run preview` — Local preview of the built site

---

## Deployment

- **Local build:**  

Output: `dist/`  
- **Vercel/Netlify:**  
- Output directory: `dist`
- For SPA routing on Vercel, add `vercel.json`:
  ```
  {
    "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }]
  }
  ```
